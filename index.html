Upon downloading the Main-files.zip file from Themeforest, you'll have all files you need inside. Zip file contains the following files

  1. Assets contains all free assets for tailoring online
  2. Demo-Content contains an XML file for demo contents.
  3. Documentation folder (It contains documentation for plugin)
  4. Plugin folder contain plugin installable files
  5. readme.txt file for general informations

To install this plugin you must have a working version of WordPress already installed. For more information regarding installing the WordPress platform, please see the WordPress Codex



Alternatively you can simply download the installable WordPress file.



When you are ready to install a plugin, you must first upload the plugin files and then activate the plugin. For this purpose, unzip the file that you downloaded from themeforest and there you would find plugin (tailors-online) file and upload it via WordPress installer. The plugin files can be uploaded in two ways

WordPress Upload

  1. Navigate to Plugins > Add New
  2. Upload Plugin and choose file then "Install Now"
  3. The plugin will be uploaded and installed
  4. Then Activate plugin


After Installation and activation you will move to plugins page.

Required Plugins

Please note : WooCommerce Plugin should be installed and activated

Import Demo Contents

To import demo contents, please follow given below steps

  1. Go to Tools > Import
  2. Install WordPress Plugin for demo import
  3. After installation Run Importer
  4. Select demo.xml which is present in downloaded packages and install it






Please note : After installation, reset permalinks.



Translation

To translate plugin any any language, please install loco translate plugin

  1. Install Loco Translate plugin
  2. Go to Loco Translate > Plugins > Tailors Online
  3. Add your language and start translations




General Settings

You can set setting for customizer.

  1. You can enable customizer option.
  2. You can add button text for customizer. Which will be shown at product detail page.
  3. You can hide or show measurements in customized order.
  4. Enable customizer at detail page will show all steps at product detail page. After selecting steps you can click on add to cart button which will attach your custom steps with order.
  5. Now you can change plugin theme color at front-end according to you choice. Just select color and save it.


When customizer will be enabled then a dropdown for all created customizers will be in shown in products. each product can have single customizer at a time.



Manage Customizer

Customizer is a full process for online customized order. For example you can create Suits, Balazers, Shirts and pants customizer

  1. Customizer need styles, either Class or Slim
  2. Customizer can have multiple apparels. Apparels mean a complete step for order. Like In Suits, blazer can be full step.
  3. Each Apparels can have two default images for front and back. For example a blazer have two views front view and back view.
  4. Each Apparels can have mutiple steps. Like A blazer can have lapels, buttons style, pockets and vents
  5. Each step have options as : IS FRONT or IS BACK, which means This step will fit at front view or back view.
  6. Each Customizer can have multiple measurements. Meaurements will be shown in checkout and cart pages.








Manage Measurements

Meaurements are for customized order. Each Customizer can have multiple measurements, each measurement have options to upload multiple images and also video options
